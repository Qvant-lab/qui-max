## Props

### name

- Type: `String`
- Default: `null`

Required prop that defines a key of QTabPane

### title 

- Type: `String`
- Default: `null`

Required prop that defines a QTabPane text

### description 

- Type: `String`,
- Default: `null`

Defines description of QTabPane

### width

- Type: `'String' | 'Number'`
- Default: `null`

Allows to configure the custom width of QTabPane

### disabled

- Type: `Boolean`
- Default: `false`

Sets disabled QTabPane state
